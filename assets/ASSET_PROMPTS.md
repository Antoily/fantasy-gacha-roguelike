## 📐 Spécifications Techniques

| Type | Dimensions | Format | Notes |
|------|-----------|--------|-------|
| Portrait héros | 80×80 | PNG transparent | Buste, face avant |
| Sprite ennemi | 60×60 | PNG transparent | Vue de face |
| Fond de salle | 360×640 | JPG/PNG | Portrait mobile |
| Icône relique | 48×48 | PNG transparent | Carré parfait |
| Cadre de carte | 100×140 | PNG transparent | UI element |
| Cellule grille | 80×80 | PNG transparent | UI tile |

**Nommage des fichiers** (à respecter pour les clés dans `BootScene.ts`) :
- Héros : `hero_{id}.png` (ex: `hero_aldric.png`)
- Ennemi : `enemy_{id}.png` (ex: `enemy_goblin_grunt.png`)
- Fond : `bg_{type}.png` (ex: `bg_combat.png`)
- Relique : `relic_{id}.png` (ex: `relic_void_crystal.png`)

**Où placer les fichiers** : `frontend/public/assets/`  
Phaser les chargera automatiquement via `this.load.image('key', 'assets/hero_aldric.png')` dans `BootScene.preload()`.
